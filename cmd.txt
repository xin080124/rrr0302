gdb forgdb

set break point
(gdb)b 3

go on after break point
(gdb)c

see the value on a specified location
gdb x 0x12345678

(gdb) print str8
$1 = 0x8048ca0 "abc"
(gdb) x 0x8048ca0

(gdb)n or s

(gdb)info stack
or 
(gdb)bt

see source code
(gdb)list

see assembly
(gdb)disas main

see frame info
(gdb)frame
